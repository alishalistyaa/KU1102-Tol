kalau kalian mau ngeview dan inspect,

1. Install Kivy
Di windows: pip install Kivy
Di MacOS: pip3 install Kivy
kalau ada kendala bisa dilihat disini: 
https://kivy.org/doc/stable/gettingstarted/installation.html

2. Install Pillow
Di windows: pip install Pillow
DI MacOS: pip3 install Pillow
kalau ada kendala bisa dilihat disini: 
https://wp.stolaf.edu/it/installing-pil-pillow-cimage-on-windows-and-mac/
